# Vision

## Проблема

Millions of old phones gather dust in drawers. Каждый из них — это 4–8 ГБ RAM, 64–256 ГБ накопителя, ARM-процессор, гарантированный uptime благодаря встроенной батарее. По мощности — больше чем Raspberry Pi 4 за $40 и многие entry-level VPS за $5/месяц.

Одновременно millions of people хотят:

- хостить свой Telegram-бот, лендинг, домашний дашборд, n8n / Vaultwarden / Pi-hole
- не платить $5/месяц за VPS на каждый pet-проект
- держать данные у себя физически
- начать в self-hosting без покупки железа

Текущие решения требуют либо Raspberry Pi + Coolify (40 USD + неделя настройки), либо VPS + Coolify ($60/год + ssh + Docker), либо энтузиастского Termux+proot setup (терминал, 2 часа боли).

## Решение

VibeFly — **один APK**. Устанавливаешь — открываешь — деплоишь сайт.

Внутри APK живёт полноценный Debian с systemd, Node, Python, PostgreSQL и нашим управляющим агентом. Пользователь работает через нативный мобильный UI с теми же фичами что у Coolify — git-deploy, env, логи, метрики, marketplace one-click apps.

Плюс AI-ассистент, который не просто пишет код (как Bolt / Lovable / v0), а **смотрит за работающим сервером**: объясняет ошибки, предлагает миграции, помогает чинить.

## Уникальное ценностное предложение

Конкуренты решают одну из задач:

- **Vercel v0, Bolt, Lovable** — генерация UI и приложений в чате, но **не** хостинг на твоём железе
- **Coolify, CapRover, Dokploy** — PaaS на твоём сервере, но **не** на телефоне и **не** AI
- **kWS, AWebServer, Andronix** — Linux на Android, но **не** PaaS и **не** для не-разработчиков

VibeFly закрывает **все три** оси: AI + PaaS + телефон.

## Целевая аудитория

1. **Self-hosters и r/homelab** (~2M активных) — основной open-source сегмент
2. **Разработчики с дорогим электричеством** (Юго-Восточная Азия, Африка, СНГ) — телефон вместо домашнего сервера
3. **Eco / repair-движение** (особенно ЕС) — против выбрасывания электроники
4. **Студенты / начинающие** — нет денег на VPS, есть старый телефон
5. **Privacy-параноики** — данные физически у себя, не в облаке

## Бизнес-модель

- **Open-source ядро** (AGPL-3.0) — агент, runtime, базовый UI
- **Cloud control plane** ($9–29/месяц) — multi-device dashboard, AI с премиум-моделями, биллинг токенов
- **Marketplace** — комиссия с платных one-click шаблонов
- **Enterprise / White-label** — позже, для переработчиков корпоративного парка телефонов

Не пытаемся конкурировать с Vercel/Bolt за $50M+ MRR. Реалистичная цель — $5–15k MRR через 12 месяцев, культовый продукт в нише self-hosting.

## Анти-цели

- **Не** заменяем VPS для критичной production-нагрузки
- **Не** генерируем приложения с нуля (это конкурентам)
- **Не** делаем DesktopOS (это PostmarketOS / Ubuntu Touch)
- **Не** делаем cloud-only (приложение должно работать без интернета на локалке)
