# Cloud control plane

Next.js приложение для управления флотом телефонов из браузера / другого устройства. **Опциональный компонент** — self-hosted VibeFly работает без него.

## Статус

Не начат. Планируется к фазе 3.

## Стек

- Next.js 15 (App Router)
- TypeScript
- Tailwind v4
- Prisma + PostgreSQL
- NextAuth v5
- Stripe

## Деплой

На Coolify, хост `cloud.vibefly.dev`.
